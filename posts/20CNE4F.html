HTTP Content-type 对照表

Content-Type，内容类型，一般是指网页中存在的 Content-Type，用于定义网络文件的类型和网页的编码，决定浏览器将以什么形式、什么编码读取这个文件，比如用 PHP 输出图片文件、JSON 数据、XML 文件等非 HTML 内容时，就必须用 header 函数来指定 Content-Type，才能达到输出一张图片或是其它指定内容类型的需求。

常用：

text/plain 　　　　　　　　纯文本类型

text/css 　　　　　　　 css 类型

text/html 　　 　　　　　html 类型

application/x-javascript 　　js 类型

application/json　　　　　 json 类型

image/png jpg gif　　　　　 image/*

multipart/form-data 上传文件 / 键值对
